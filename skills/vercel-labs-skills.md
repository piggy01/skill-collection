## 📝 简介

`npx skills` 跨 Agent 技能安装/管理/搜索 CLI 工具，支持 67 种 AI 编码 Agent。

## 📦 已安装的技能

| 技能 | 说明 |
|---|---|
| `find-skills` | 搜索和发现可用技能 |

## 🔧 常用命令

```bash
npx skills add <source>     # 安装技能
npx skills list              # 列出已安装
npx skills find [query]     # 搜索技能
npx skills remove <skill>   # 移除技能
npx skills update [skill]   # 更新技能
```
